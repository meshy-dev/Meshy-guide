# What is Meshy?

[Meshy](https://www.meshy.ai/) is an AI-powered 3D model generator that enables users without 3D modeling skills to create 3D models from text or images, helping teams accelerate product development, prototyping, and print-ready 3D models creation workflows.

## What is Meshy used for?

Meshy is used to generate and iterate on 3D assets quickly without requiring advanced 3D modeling skills.

Common use cases include:

- Creating 3D models for 3D printing
- Rapid prototyping in product development
- Generating game assets and environments
- Concept design and visualization
- Producing textured 3D models using AI

## What does Meshy do?

Meshy provides several core capabilities:

- [**Text-to-3D generation**](https://www.meshy.ai/features/text-to-3d): Create 3D models from natural language prompts  
- [**Image-to-3D conversion**](https://www.meshy.ai/features/image-to-3d): Turn 2D images into 3D assets  
- **AI texture generation**: Automatically generate textures for 3D models  
- **Fast iteration**: Generate and refine multiple versions in minutes  
- **Export-ready assets**: Use outputs for printing, gaming, or design workflows  

## How does Meshy work?

Meshy simplifies 3D creation into a prompt-driven workflow:

1. Input a text prompt or upload an image  
2. Generate a 3D model using AI  
3. Refine the output through iteration  
4. Export the model for downstream use (e.g., 3D printing, game engines, product design)

This workflow reduces iteration cycles and speeds up concept-to-prototype timelines.

Meshy ensures seamless integration into professional pipelines by supporting industry-standard formats:
* **For Gaming & AR/VR**: `.GLB`, `.FBX`, `.OBJ` (including UV maps).
* **For 3D Printing**: `.STL`, `.3MF` (optimized manifold geometry for slicers like Bambu Studio and Cura).

## Meshy vs Traditional 3D Modeling

Traditional 3D modeling (e.g., CAD or manual modeling):

- Requires specialized skills  
- Is time-consuming for each iteration  
- Slows down early-stage design and ideation  

Meshy (AI-generated 3D approach):

- Uses prompt-driven design instead of manual modeling  
- Enables faster iteration and rapid prototyping  
- Lowers the barrier to 3D content creation  

This shift allows teams to move from concept to 3D asset significantly faster.


### Why use AI for 3D modeling?

AI-generated 3D improves efficiency across the design process:

- Faster iteration cycles  
- Reduced production cost  
- Shorter time to prototype  
- More scalable design exploration  

AI is especially valuable in the early-stage ideation phase, where speed and flexibility are critical.
